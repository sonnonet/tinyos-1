Component: tos.chips.ms5607.ResourceReadRefC

generic module ResourceReadRefC(typedef width_t)

Implement arbitrated access to a Read interface, based on an underlying arbitrated Resource interface. Note that this code does not deal with unexpected events: it assumes that all events it receives are in response to commands that it made. See tos/chips/atm128/adc for an example of using ArbitratedReadC in a safe way.

Parameters:
width_t - Width of the underlying Read interface.
